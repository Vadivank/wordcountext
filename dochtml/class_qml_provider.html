Detailed Description

QmlProvider - класс, представлюящий собой масштабируемый слой-прокладку между QML и C++ логикой.

Поскольку приложение обладает довольно простым набором контроля и для простоты реализации, коннекты метаобъектной системы Qt помщенны в данном классе, а не в отдельном.
